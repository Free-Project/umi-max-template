# README

`@umijs/max` template project. For more features, refer to [Umi Max Introduction](https://umijs.org/docs/max/introduce).


- 📦 Ready-to-use Admin Template – Dashboard, CRUD tables, access control examples
- 🎨 Pro Components – ProTable, ProForm, ProDescriptions from @ant-design/pro-components
- 🔐 Access Control – Built-in permission management system
- 🌐 Request Layer – Integrated request configuration with interceptors
- 📱 Responsive Layout – Custom MaxLayout with sidebar navigation
- 🛠️ Development Experience – ESLint, Prettier, Husky pre-commit hooks
